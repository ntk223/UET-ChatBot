Dưới đây là toàn bộ tài liệu đặc tả dự án **Chatbot Hỗ trợ Tuyển sinh & Đăng ký Nguyện vọng** dựa trên kiến trúc **Flowchart - Rasa System** đã được tổng hợp, đồng bộ hóa và làm sạch hoàn chỉnh.

Tài liệu này tích hợp đầy đủ cả 4 phương thức xét tuyển: **HSA, IELTS kết hợp, Tuyển thẳng, và Điểm thi THPTQG (kèm minh chứng URL Cloud)**. Nhóm của Kiên có thể dùng bản này làm tài liệu kỹ thuật chính thức để triển khai.

---

## 1. Kiến trúc tổng quan hệ thống (System Architecture)

Hệ thống hoạt động theo mô hình Hybrid: **Hội thoại tự do (NLU/FAQ)** ở giai đoạn tìm hiểu thông tin và chuyển tiếp sang **Hội thoại định hướng (Form/State Machine)** bằng các nút bấm điều hướng khi thí sinh đăng ký nguyện vọng.

```text
[Frontend: Web/Facebook Chat] 
       │ (Gửi câu hỏi / Payload nút bấm)
       ▼
[Rasa Open Source Server] ──(Nếu là FAQ)──► Trả về thông tin ngay (Học phí, Điểm chuẩn)
       │ (Kích hoạt Form tương ứng phương thức xét tuyển)
       ▼
[Rasa Action Server (Python)]
       │ (Thực thi Custom Actions / Xử lý Logic Form)
       ▼
[Database: MySQL Server] ──► (Lưu trữ thông tin & Minh chứng URL)

```

## 2. Thiết kế Cơ sở dữ liệu (MySQL Schema)

Cấu trúc gồm 1 bảng danh mục (`majors`), 1 bảng định danh thí sinh chung (`candidates`), và 4 bảng phụ tương ứng với các nhánh flowchart thu thập thông tin đặc thù.

```sql
CREATE DATABASE IF NOT EXISTS university_admission CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE university_admission;



-- 1. Danh mục Ngành học để bot tra cứu dữ liệu động

CREATE TABLE majors (

    major_code VARCHAR(20) PRIMARY KEY,

    major_name VARCHAR(100) NOT NULL,

    tuition_fee DECIMAL(10, 2),

    benchmark_2025 FLOAT,

    quota INT,

    description TEXT

);



-- 2. Bảng định danh thông tin chung của thí sinh

CREATE TABLE candidates (

    id INT AUTO_INCREMENT PRIMARY KEY,

    fullname VARCHAR(100) NOT NULL,

    phone_number VARCHAR(15) NOT NULL,

    chosen_major_code VARCHAR(20),

    admission_method VARCHAR(30) NOT NULL, -- 'HSA', 'IELTS', 'TUYEN_THANG', 'THPTQG'

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (chosen_major_code) REFERENCES majors(major_code)

);



-- 3. Bảng phụ: Phương thức HSA (Đánh giá năng lực)

CREATE TABLE admission_hsa (

    candidate_id INT PRIMARY KEY,

    hsa_id VARCHAR(50) NOT NULL,

    hsa_score FLOAT NOT NULL,

    evidence_url VARCHAR(500) NOT NULL,   -- Link ảnh minh chứng lưu trên Cloud

    FOREIGN KEY (candidate_id) REFERENCES candidates(id) ON DELETE CASCADE

);



-- 4. Bảng phụ: Phương thức IELTS kết hợp

CREATE TABLE admission_ielts (

    candidate_id INT PRIMARY KEY,

    ielts_score FLOAT NOT NULL,

    highschool_math_score FLOAT NOT NULL,

    evidence_url VARCHAR(500) NOT NULL,   -- Link ảnh minh chứng lưu trên Cloud

    FOREIGN KEY (candidate_id) REFERENCES candidates(id) ON DELETE CASCADE

);



-- 5. Bảng phụ: Phương thức Tuyển thẳng

CREATE TABLE admission_direct (

    candidate_id INT PRIMARY KEY,

    award_name VARCHAR(255) NOT NULL,

    award_rank VARCHAR(50) NOT NULL,

    evidence_url VARCHAR(500) NOT NULL,   -- Link ảnh minh chứng lưu trên Cloud

    FOREIGN KEY (candidate_id) REFERENCES candidates(id) ON DELETE CASCADE

);



-- 6. Bảng phụ: Phương thức điểm thi THPTQG (Thang điểm 30 + Link Cloud)

CREATE TABLE admission_thptqg (

    candidate_id INT PRIMARY KEY,

    block_name VARCHAR(10) NOT NULL,       -- Khối thi: A00, A01, D01...

    total_score FLOAT NOT NULL,           -- Tổng điểm 3 môn

    evidence_url VARCHAR(500) NOT NULL,   -- Link ảnh minh chứng lưu trên Cloud

    FOREIGN KEY (candidate_id) REFERENCES candidates(id) ON DELETE CASCADE

);
```

## 6. Logic Backend Xử lý kết nối Database (`actions/actions.py`)

```python
from typing import Any, Text, Dict, List
from rasa_sdk import Action, Tracker
from rasa_sdk.executor import CollectingDispatcher
import mysql.connector

def get_db_connection():
    return mysql.connector.connect(
        host="localhost",
        user="root",
        password="your_password",
        database="university_admission"
    )

# --- GIAI ĐOẠN 1: TRA CỨU ĐỘNG THÔNG TIN ---
class ActionQueryMajorInfo(Action):
    def name(self) -> Text:
        return "action_query_major_info"

    def run(self, dispatcher: CollectingDispatcher, tracker: Tracker, domain: Dict[Text, Any]) -> List[Dict[Text, Any]]:
        nganh_hoc = next(tracker.get_latest_entity_values("nganh_hoc"), None)
        if not nganh_hoc:
            dispatcher.utter_message(text="Bạn muốn hỏi về ngành cụ thể nào? Ví dụ: Khoa học máy tính.")
            return []
        try:
            conn = get_db_connection()
            cursor = conn.cursor(dictionary=True)
            cursor.execute("SELECT * FROM majors WHERE major_name LIKE %s", (f"%{nganh_hoc}%",))
            major = cursor.fetchone()
            if major:
                response = (f"Ngành {major['major_name']} ({major['major_code']}):\n"
                            f"- Học phí: {major['tuition_fee']} VNĐ/kỳ.\n"
                            f"- Điểm chuẩn 2025: {major['benchmark_2025']}\n"
                            f"- Chỉ tiêu: {major['quota']}\nBạn có muốn nộp hồ sơ xét tuyển không?")
                dispatcher.utter_message(text=response, buttons=[{"title": "Đăng ký ngay", "payload": "/dang_ky_nguyen_vong"}])
            else:
                dispatcher.utter_message(text=f"Hiện tại mình chưa tìm thấy thông tin ngành {nganh_hoc}.")
            cursor.close()
            conn.close()
        except Exception as e:
            dispatcher.utter_message(text="Hệ thống tra cứu đang bận, thử lại sau!")
        return []

# --- GIAI ĐOẠN 2: SUBMIT FORM THPTQG ---
class ActionSubmitThptqgForm(Action):
    def name(self) -> Text:
        return "action_submit_thptqg_form"

    def run(self, dispatcher: CollectingDispatcher, tracker: Tracker, domain: Dict[Text, Any]) -> List[Dict[Text, Any]]:
        slots = ["fullname", "phone_number", "chosen_major", "thptqg_block", "thptqg_score", "evidence_url"]
        data = {s: tracker.get_slot(s) for s in slots}
        try:
            conn = get_db_connection()
            cursor = conn.cursor()
            
            # Ghi vào bảng gốc candidates
            cursor.execute("""
                INSERT INTO candidates (fullname, phone_number, chosen_major_code, admission_method)
                VALUES (%s, %s, (SELECT major_code FROM majors WHERE major_name LIKE %s LIMIT 1), 'THPTQG')
            """, (data["fullname"], data["phone_number"], f"%{data['chosen_major']}%"))
            candidate_id = cursor.lastrowid
            
            # Ghi vào bảng phụ admission_thptqg kèm link Cloud URL
            cursor.execute("""
                INSERT INTO admission_thptqg (candidate_id, block_name, total_score, evidence_url)
                VALUES (%s, %s, %s, %s)
            """, (candidate_id, data["thptqg_block"], data["thptqg_score"], data["evidence_url"]))
            
            conn.commit()
            cursor.close()
            conn.close()
            dispatcher.utter_message(text=f"Chúc mừng {data['fullname']} đã đăng ký thành công! Mã hồ sơ: #{candidate_id}. Ban tuyển sinh sẽ thẩm định minh chứng tại link bạn đã gửi.")
        except Exception as e:
            dispatcher.utter_message(text="Gặp sự cố khi lưu hồ sơ THPTQG trực tuyến.")
        return []

```

Bản tổng hợp đặc tả này đã sẵn sàng để phân phối. Việc đồng bộ chặt chẽ cấu trúc tên `slot` của Rasa với cột cơ sở dữ liệu `MySQL` sẽ giúp bạn Backend phát triển các Custom Action còn lại (`action_submit_hsa_form`, `action_submit_ielts_form`) một cách rất dễ dàng dựa trên khuôn mẫu của hàm THPTQG ở trên.

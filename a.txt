TÀI LIỆU THIẾT KẾ KIẾN TRÚC HỆ THỐNG
Dự án: Chatbot Tư vấn Tuyển sinh Đại học (Flowchart-grounded AI)
1. Kiến trúc tổng thể (High-Level Architecture)
Hệ thống được thiết kế theo mô hình Decoupled Architecture, tách rời phần hiểu ngôn ngữ (NLU) và phần điều phối hội thoại (Orchestration).
Client Layer: Giao diện người dùng (Web/Mobile), gửi và nhận tin nhắn qua Socket.io hoặc REST API.
Orchestrator Layer (Node.js): Đóng vai trò là "Tổng đài viên", điều phối luồng dữ liệu giữa NLU, Database và Client.
NLU Layer (Rasa): Dịch vụ chuyên biệt để phân tích ý định (Intent) và thực thể (Entity).
Data Layer: PostgreSQL (Dữ liệu nghiệp vụ) và Redis (Quản lý trạng thái/Session).

2. Các thành phần cốt lõi (Core Components)
2.1. Node.js Orchestrator (Dialogue Manager)
Đây là trung tâm xử lý logic của hệ thống.
Flowchart Parser: Đọc và ánh xạ file flowchart.json vào bộ nhớ.
State Router: Quyết định Node tiếp theo dựa trên kết quả từ Rasa và trạng thái hiện tại trong Redis.
Integrator: Gọi các API bên ngoài hoặc truy vấn Database để lấy dữ liệu động (điểm chuẩn, học phí).
2.2. Rasa NLU Service
Đóng vai trò là một "Pure NLU Service".
Pipeline: DIETClassifier, EntitySynonymMapper, ResponseSelector.
Output: Trả về JSON chứa Intent (độ tin cậy > 0.8) và các Entities đã chuẩn hóa.
2.3. Lớp lưu trữ (Storage Layer)
Thành phần
Công nghệ
Nội dung lưu trữ
Business DB
PostgreSQL
Thông tin ngành, mã ngành, điểm chuẩn, học phí, học bổng.
State Store
Redis
current_node, slots_filled, conversation_history.
Metadata
JSON/YAML
Cấu trúc đồ thị Flowchart (Nodes, Edges, Conditions).


3. Thiết kế luồng xử lý (Sequence Flow)
Quy trình xử lý một câu lệnh từ người dùng được thực hiện qua các bước sau:
Input: Client gửi câu hỏi: "Ngành KHMT lấy bao nhiêu điểm?".
Understand: Node.js gửi text sang Rasa. Rasa trả về: {intent: "ask_score", entities: {major: "Khoa học máy tính"}}.
Map: Node.js tra cứu flowchart.json, thấy Intent ask_score dẫn đến Node diem_chuan.
Fetch: Node.js lấy mã ngành CN1 từ bảng mapping, query PostgreSQL lấy điểm chuẩn năm 2025.
Render: Node.js kết hợp dữ liệu vào template câu trả lời và gửi kèm các nút bấm (e.g., "Xem học phí", "Cơ hội việc làm").
Update: Cập nhật current_node = diem_chuan vào Redis.
